# ProphetProject
This project is predict Long term Prediction the market demand for printing industry
terdapat dua file PNG yaitu:

a) prophet_forecast_plot.png
Ini adalah ringkasan visual yang paling kuat. Terdapat Tren (apakah permintaan tumbuh atau stagnan), Musiman (kapan puncak dan lembah permintaan terjadi), dan Jeda Waktu Prediksi (garis putus-putus merah).
Tujuannya	Membantu Manajer dan Direktur membuat keputusan strategis dengan melihat big picture dari masa depan permintaan pasar.

b) prophet_components_plot.png
*Tren:** Menunjukkan pertumbuhan atau penurunan permintaan jangka panjang.
* **Musiman Tahunan:** Mengidentifikasi bulan/kuartal mana yang memiliki permintaan paling tinggi (misalnya, Q4 karena promo akhir tahun).
* **Musiman Mingguan:** Menunjukkan hari-hari (misalnya, Selasa/Rabu) di mana perusahaan harus menyiapkan kapasitas ekstra. |
